# STL File Organizer (Private)

Canonical: https://stl-shelf.com/stl-file-organizer

Organize STL files in a private library with tags, 3D preview, and version history. No marketplace, no sharing - just your files.
Use a private STL organizer that focuses on clarity: tags, preview, and version history without marketplace noise.

## The problem

- STL collections usually grow faster than folder systems.
- After a few months, similar file names and duplicated exports make retrieval slow.
- Without a structured workflow, critical revisions disappear at print time.

## Simple workflow

1. **Move STL files into one private library**: Store all models in one place so every search starts from a single source of truth.
2. **Apply focused tags**: Tag by project, part, or status to find exactly what you need in seconds.
3. **Use preview plus version history**: Inspect models quickly and track each change with a clean revision timeline.

## What STL Shelf is / is not

### STL Shelf is

- A private library for your STL, 3MF, and OBJ files
- Searchable tags that keep projects and parts organized
- Version history so every revision stays traceable
- Browser preview before you send models to print

### STL Shelf is not

- No marketplace
- No sharing or social feed
- No import or sync from other services

## FAQ

### Is this a public repository?

No. STL Shelf is private and centered on your personal files.

### Can teams publish models from STL Shelf?

No. STL Shelf is not a publishing tool or marketplace.

### Why use tags if I already have folders?

Tags let one model belong to multiple contexts without duplicating files.

### Can I compare revisions later?

Yes. Version history keeps every STL revision available for reference.
