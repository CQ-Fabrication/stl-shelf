# 3D model preview in the browser

Canonical: https://stl-shelf.com/3d-model-preview-in-browser

Preview 3D models in your browser to quickly inspect files before printing. Keep everything organized in a private library.
Browser preview helps you validate files before slicing, so you can catch mistakes early and avoid wasted print cycles.

## The problem

- When preview is separated from your file library, inspection gets skipped.
- That increases the chance of printing the wrong revision.
- Fragmented tools also make file handoff and review slower.

## Simple workflow

1. **Open models directly in browser preview**: Check geometry quickly without downloading or switching tools.
2. **Tag and categorize while reviewing**: Add tags during inspection so files stay organized and searchable.
3. **Tie preview to version history**: Review the exact revision you plan to print, not a guess from folders.

## What STL Shelf is / is not

### STL Shelf is

- A private library for your STL, 3MF, and OBJ files
- Searchable tags that keep projects and parts organized
- Version history so every revision stays traceable
- Browser preview before you send models to print

### STL Shelf is not

- No marketplace
- No sharing or social feed
- No import or sync from other services

## FAQ

### Does preview support STL, 3MF, and OBJ?

Yes. STL Shelf supports browser preview across these core model formats.

### Can preview replace slicing?

No. Preview is for fast inspection before slicer preparation.

### Why connect preview and versions?

It ensures you inspect the same revision you intend to print.

### Is preview public?

No. STL Shelf is a private file library with no social sharing.
