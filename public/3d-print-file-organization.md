## A mixed-format organization workflow

1. **Centralize all supported formats**: Keep STL, 3MF, OBJ, and PLY assets in one private archive instead of maintaining separate storage rules for each format.
2. **Organize by model and purpose**: Treat the printable item as the main unit and attach its related files, versions, and tags under one organizational umbrella.
3. **Retrieve by outcome**: Search for the model you need and then choose the correct file variant or version from that record.

## Why multi-format libraries are harder to manage

Mixed-format libraries break folder systems faster because different file types tend to enter the archive from different tools and moments in the workflow. If each format lives in a different place, the model loses cohesion.

## FAQ

### Does STL Shelf support STL only?

No. STL Shelf supports STL, 3MF, OBJ, and PLY files in the same private library workflow.

### Why is 3D print file organization broader than STL organization?

Because many libraries include multiple formats tied to the same model or project. Organizing only one format leaves the archive fragmented.

### Can STL Shelf manage a mixed-format archive?

Yes. Mixed-format support is part of the product positioning.

### Is STL Shelf a publishing platform?

No. It is software for private archive management, not publishing or selling.
