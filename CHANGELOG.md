# Changelog

All notable changes to STL Shelf will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-09

### Added

- Initial open source release under Apache 2.0 license
- 3D model management (upload, organize, search, tag)
- Interactive 3D preview for STL, OBJ, 3MF, and PLY files
- Version history for models with file tracking
- Organization support with team member management
- Role-based access control (Owner, Admin, Member)
- Subscription tiers via Polar.sh billing
- Self-hosting support with Docker
- S3-compatible storage backend (R2, MinIO, AWS S3)
- Magic link and OAuth authentication (GitHub, Google)
- Terms & Privacy acceptance with consent audit trail
- Responsive design for desktop and mobile
